# weather-data-analytics-platform
Weather Data Analytics Platform

This project demonstrates an end-to-end data engineering pipeline including:

• Python ETL (CSV → MySQL)
• MySQL Star Schema Data Warehouse
• Advanced SQL (Window Functions, Ranking, Indexing)
• Spark Batch Processing
• Spark Streaming Simulation
• Snowflake Data Warehouse Implementation
• Role-Based Access Control & Data Masking
• Streamlit Dashboard

Tech Stack

Python, MySQL, PySpark, Streamlit, Snowflake

Execution Order

Run weather_warehouse.sql

Run etl.py

Run dashboard.py

Run spark_processing.py

Run spark_streaming.py
